Network Categories

Peer-to-peer Network

  • Devices share the same resources and privileges without a centralized administrative system
  • They can connect and make requests directly with one another

For instance, BitTorrent (BT) is a communication protocol for P2P file sharing. It is commonly used in downloading large files, such as videos and Linux installation files.


Client/server Networks

  • A centralised administrative system known as the server is present
  • Shares files and resources to clients connected to it

Network Topologies


Network topology refers to the layout of a network. Physical topology describes how devices in a network are wired together, while logical topology decribes how data is transmitted within a network.


  1. Ring Topology : If one device stops working, the network won't work properly
  2. Star Topology : If on device(not the central device) stops working, the network won't be affected
  3. Bus topology : If the bus is damaged then the whole network fails
  4. Mesh topology : A failure of a single node doesn't affect the network connectivity. However, the cost is high and it is difficult to set up
